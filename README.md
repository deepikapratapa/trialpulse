# TrialPulse — Clinical Trial Operations & Risk Analytics (ClinicalTrials.gov)

TrialPulse is an analytics-first decision-support project built on ClinicalTrials.gov data to help clinical operations and research leaders quantify timelines, enrollment risk, discontinuation patterns, sponsor differences, and geographic distribution across Phase II–III trials.

## Planned Outputs
- Reproducible pipeline: download → clean/standardize → analysis-ready dataset
- 7–10 executive-ready figures in `reports/figures/`
- Summary tables in `reports/tables/`
- Streamlit dashboard with filters, KPI cards, risk insights, and downloads

## Data Source
ClinicalTrials.gov (official API v2)
